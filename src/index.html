Picon

Picon はあなたのAndroid端末へプッシュメッセージを配信するAPIを提供します。

要件

  • Android 2.2以上を搭載した端末
  • Googleアカウント (今のところGoogle Appsアカウントでは動作しません)
  • Picon Android Client アプリ

使い方

  1. Piconサーバにサインインします
  2. Android端末のPiconクライアントにより端末の登録をします
  3. Piconサーバのダッシュボードからテストメッセージを送ります
  4. APIをつかってメッセージを送信してみます。

API

POST https://c2dmsend.appspot.com/api/1/send
key: API_KEY
event: event
text: message
url: additional URL
priority: priority
